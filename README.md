

Titre : Simulateur de course de trot

Auteur : Séverine Hori Maitrehut

Description : 
Ce programme simule une course de chevaux. 
À la fin de la course, les chevaux sont classés selon leur ordre de franchissement de la ligne d'arrivée.

Structures de données choisies :

La liste des chevaux est un tableau de dictionnaires. Chaque cheval est représenté par un dictionnaire avec les informations nécessaires : num_horse, speed, distance_traveled, disqualified et turn_arrival

J'ai créé un tuple pour pouvoir connaître la distance suivant la vitesse. L'indice correspond à la vitesse, et la valeur correspond à la distance

Pour pouvoir modifier la vitesse suivant le lancer de dés j'ai opté pour un tableau à deux dimensions.
